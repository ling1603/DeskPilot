# 截图占位说明

README 中预留了截图位置，建议后续放入以下文件：

- `main-window.png`：主窗口整体界面
- `rules-editor.png`：分类规则编辑器
- `tray-menu.png`：系统托盘菜单

- 主窗口截图：约 1200 × 800
- 局部功能截图：约 900 × 600
- 文件夹效果截图：约 1200 × 700
